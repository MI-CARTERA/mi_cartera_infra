# mi_cartera_infra
Infrastructure for 'Mi Cartera' project.

# Proceso de desarrollo

1) Definir la arquitectura base del proyecto, basado en los requerimientos y funcionalidades.

2) Eleccion de las tecnologías, de acorde al objetivo del proyecto e implementación, de la mano de otras tecnologías.

3) Definir la estetica de la interfáz de la web/aplicación

4) Levantar la arquitectura de la red del proyecto
